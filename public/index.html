# Projects

Rif

Rif is an impact control system. Rif enables users to control impacts of file changes which has deep inter-connectivity with other files.

Gde

Gde is an game design document creation engine. Gde can yield various formats such as pptx, docx or pdf from single input gddt file.

R4d

R4d(rad) is a text oriented macro processor. R4d was created as an alternative to m4 macro processor. There are no m4 quotes neither no ambiguous macro calls. R4d can be used as a library and highly configurable.

# Reverse designs

I've made several reverse design documents of games. Every documents were made with gdengine(or its predecessor gdmarp). Mostly projects' target platform is pdf slide.
